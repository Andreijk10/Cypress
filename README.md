# Cypress

Cypress Automation Projects
This repository documents my journey of learning and applying Cypress for web automation testing. It includes hands-on lessons, practice exercises, and real automated test cases built to strengthen my testing skills.

Automated Test Scenarios
Developed and executed automated test suites for the Mindomo web application, covering key user workflows:
-Export functionality
-Print feature
-Sign-up process
-Purchase subscription flow

Structure & Resources
-Organized classes for better code reusability and test maintenance
-JSON data files for input handling and test validation
-Modular test design following best practices in Cypress

Learning Materials from courses: Testare manuala si automata (Curs QA) - by RaduPopescu and The complete Cypress 13+ Course - by Joan Montero
-Step-by-step lessons on Cypress fundamentals
-Practice examples for UI interactions, assertions, and test organization
